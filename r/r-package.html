read_csv() 函数

  • c: 字符（character）
  • i: 整数（integer）
  • n: 数字（number，双精度浮点数）
  • d: 日期（date）
  • t: 时间（time）
  • T: 日期时间（date time）
  • l: 逻辑值（logical，TRUE 或 FALSE）
  • _ 或 -: 跳过该列（不导入）
  • ?: 自动推断类型（默认行为） 
# 创建一个 CSV 文件示例
csv_content <-

	Cathy,28,168.2,TRUE"

cols() 函数

根据列名指定某几列的数据类型。cols() 函数允许你通过列名来定义列类型，而不是通过位置定义。

# 安装并加载 readr 包
if (!requireNamespace("readr", quietly = TRUE)) {
  install.packages("readr")
}
library(readr)

# 创建一个 CSV 文件示例
csv_content <- "Name,Age,Height,IsStudent
Alice,30,160.5,TRUE
Bob,25,175.0,FALSE
Cathy,28,168.2,TRUE"

# 使用 cols() 函数根据列名指定列类型
data <- read_csv(csv_content, 
                 col_types = cols(
                     Name = col_character(),
                     Age = col_integer(),
                     Height = col_double(),
                     IsStudent = col_logical()
                     )
                 )

# 打印导入的数据
print(data)
