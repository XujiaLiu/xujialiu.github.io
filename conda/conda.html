常用的 conda 命令：

  1. 创建新环境：
conda create --name myenv

你可以指定 Python 版本以及所需的包：

conda create -n myenv python=3.6 numpy pandas
  1. 激活环境：
conda activate myenv
  1. 取消激活环境：
conda deactivate
  1. 列出所有环境：
conda env list
conda info -e
  1. 删除环境：
conda env remove --name myenv
  1. 安装包：
conda install numpy

你也可以指定包版本：

conda install numpy=1.15.1
  1. 更新包：
conda update numpy
  1. 卸载包：
conda remove numpy
  1. 列出环境中的包：
conda list
  1. 查找包：
conda search numpy
  1. 更新 conda：
# 更新conda repodata.json
conda update --all
codna update conda

anaconda通道和conda-forge通道

  • conda create -c conda-forge -n r-4.2 r-base=4.2.3：
    • 这条命令会在conda-forge这个通道中搜索并安装 r-base 版本 4.1.2。
    • conda-forge是一个由社区维护的Conda通道，它包含了许多官方主通道（默认的anaconda通道）没有的包。使用conda-forge通道往往可以获取到最新的软件包版本。
  • conda install r-base：
    • 这条命令会在你配置的默认通道中搜索并安装 r-base 版本 。
    • 如果你没有特别指定通道，那么这条命令通常会在Anaconda的默认通道中搜索这个包。

总的来说，两条命令的区别在于它们在哪个Conda通道中搜索和安装包。conda-forge通道往往包含了更多的包和更新的版本，但在某些情况下，使用默认的Anaconda通道可能更稳定或更方便。你可以根据你的需要选择使用哪个通道。

关闭自动激活base环境 

conda config --set auto_activate_base false
