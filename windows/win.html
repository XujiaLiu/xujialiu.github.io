windows挂载网盘打开http协议

windows默认只支持https协议挂载硬盘, 如果需要用http协议挂载webdav, 需要修改注册表

  1. 修改regedit.exe注册列表
  2. Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WebClient\Parameters
  3. BasicAuthLevel值改为2
  4. cmd重启webclient服务：net stop webclient和 net start webclient

rclone mount xujialiu_zspace:/ Y: --cache-dir C:/cache/rclone --vfs-cache-mode writes

Option Explicit
Dim WMIService, Process, Processes, Flag, WS
Set WMIService = GetObject("winmgmts:{impersonationlevel=impersonate}!\\.\root\cimv2")
Set Processes = WMIService.ExecQuery("select * from win32_process")
Flag = true
for each Process in Processes
    if strcomp(Process.name, "rclone.exe") = 0 then
        Flag = false
        exit for
    end if
next
Set WMIService = nothing
if Flag then
    Set WS = Wscript.CreateObject("Wscript.Shell")
    WS.Run "rclone mount xujialiu_zspace:/ Y: --cache-dir C:/cache/rclone --vfs-cache-mode writes", 0
end if

还原成win10

  • 桌面右键打开cmd
  • 输入: reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
  • 打开注册列表的HKCU\Software\Classes\CLSID{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32，点击一下文件夹里的项目后直接点确认
  • 重启
    还原成win11
  • 打开cmd
  • 输入reg.exe delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f

更新win11的问题

无法判断你的电脑是否有足够的空间继续安装Windows11.请尝试重新启动安装程序

  • 如果是动态卷(dynamic disk)是不能更新win11 22H2, 需要将dynamic disk先转换成basic disk才能更新win11
  • 更改需要用到easeUS这款软件EaseUS
  • 软件更改卷时, 会多次重启
  • 更改好了再重新启动更新镜像即可

pdf的地位

pdf的好处:

  • 支持好, 大量软件都可以打开pdf格式的文件
  • 应用广, 几乎所有的出版物都有pdf版本
  • zotero和obsidian可以完美支持

pdf的工作流:

  • 所有的书籍, 包括文献都慢慢过度到用zotero进行管理
  • 用zotero的pdf阅读器进行阅读和做笔记
  • 用obsidian和zotero对笔记和要点进行整理
  • 对于没有pdf版本的书籍, 统一用calibra转换成pdf版本
