逻辑替换

where()和 mask()
where()在传入条件为 False 的对应 行进行替换, 而 mask()在传入条件为 True 的对应行进行替换, 当未对二者指定替换值时, 将对应 行替换为缺失值。

s.where(s<0)
s.where(s<0,  100)

s.mask(s<0)
s.mask(s<0,  -50)

s_condition= pd.Series([True, False, False, True], index=s.index)
s.mask(s_condition,  -50)

窗口

3 类窗口, 分别是滑动窗口 rolling, 扩张窗口/累计窗口 expanding 以及指数加权窗口 ewm

滑动窗口

rolling() 

s = pd.Series([1, 2, 3, 4, 5])
roller = s.rolling(window = 3)
roller.mean()

shift(), diff()和 pct_change()

公共参数为 periods=n, 默认值为 1, n 为负值, 表示反方向进行类似操作

  • shift: 它们分别表示取向前第 n 个元素的值
  • diff: 与向前第 n 个元素作差和
  • pct_change: 与向前第 n 个元素相比而计算出的增长率

扩张窗口

动态长度窗口，其窗口的长度就是从序列开始处的位置到具体操作处的位置，其使用的聚合函数会作用于这些逐步扩张的窗口。具体地说，设序列为[a1,a2,a3,a4]，则其每个位置对应的窗口即[a1]、[a1,a2]、[a1,a2,a3]、[a1,a2,a3,a4], 即第n为为前n个元素所构成的窗口
在 NumPy 中，cumsum()和 cumprod()分别计算当前位置与之前所有元素的总和与乘积
利用 expanding 对象上的 apply()方法来等价地完 成上述操作：

s.expanding().apply(lambda x: x.sum())
s.expanding().apply(lambda x: x.prod())

  • Series 的行索引

		 ((School == "A") & (Grade == "Senior") & (Weight > 70)) |
		 ((School == "B") & (Grade!= "Senior") & (Weight > 80))
		 """)
		 
# 对于含有空格的列名，需要使用`col name`的方式进行引用，例如 
df.query('`col name`=="A"')

df.query('(Grade not in ["Freshman", "Sophomore"]) and (Gender=="Male")')

# 引用外部变量

多级引索 

df_ex.loc[pd.IndexSlice['C':, ('D', 'f'):]]

# 布尔索引
df_ex.loc[pd.IndexSlice[:'A', lambda x:x.sum()>0]] # 列和大于 0

# loc[pd.IndexSlice[...,...],pd.IndexSlice[...,...]]型的 slice 对象能够进行分层切片
# 前一个 pd.IndexSlice 指代的是行索引，后一个 pd.IndexSlice 指代的是列索引
df_ex.loc[pd.IndexSlice[:'A', 'b':], pd.IndexSlice['E':, 'e':]]

索引层的交换和删除

  • swaplevel()和 reorder_levels()完成，前者只能交换两层，而后者可以交换任意层，两者都可以指定交换的“轴”，即指定交换行索引或交换列索引
    origin
df_ex.swaplevel(0,2,axis=1)



df_ex.reorder_levels([2,0,1],axis=0)

  • droplevel() 删除某一层的索引 
df_ex.droplevel([0,1],axis=0)
